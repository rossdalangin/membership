1. Installation & Setup

Installation

Installing WordPress Membership Pro is the same as any other WordPress plugin:

  1. Download the plugin's .zip file.
  2. From your WordPress dashboard, navigate to Plugins → Add New.
  3. Click the Upload Plugin button at the top of the page.
  4. Choose the downloaded .zip file and click Install Now.
  5. Once installed, click Activate Plugin.

Initial Setup

After activation, you should configure your payment gateways.

  1. Navigate to Membership Plans → Settings.
  2. Under the "Payment Gateways" section, enter your API keys for PayPal or other enabled gateways.
  3. Configure any other settings on this page as needed.
  4. Click Save Changes.

2. Creating Essential Pages (Shortcodes)

The plugin uses shortcodes to display dynamic content on the front-end. You need to create several pages and place the corresponding shortcodes on them.

Page Purpose Recommended Page Name Shortcode to Use
Display your membership plans for purchase. Plans [wmp_plans]
Handle the final step of the purchase process. Checkout [wmp_checkout]
Allow users to view their subscriptions and billing history. My Account [wmp_account]
Display a thank you message after a purchase. Thank You [wmp_thank_you]
Display a special upsell or downsell offer. One-Time Offer [wmp_oto]
Allow users to sign up for your affiliate program. Affiliate Program [wmp_affiliate_registration]
Allow affiliates to view their stats and referral link. Affiliate Dashboard [wmp_affiliate_dashboard]

3. Managing Your Site

3.1. Managing Customers & Subscribers

All active and past subscriptions are listed in the WordPress admin area.

  • List of Subscribers: Navigate to Subscriptions from the main admin menu. This screen lists all subscriptions from all customers.
  • List of All Transactions: Navigate to Subscriptions → Transactions.
  • Communicating with Customers: The plugin currently does not have a built-in email system. We recommend using a dedicated email marketing plugin and integrating it with your member roles. You can find user email addresses by navigating to the main WordPress Users menu.

3.2. Managing Content (Restriction & Drip)

You can protect your posts and pages and schedule their release for members.

  1. Edit any post or page.
  2. On the right-hand sidebar, you will find the Membership Access meta box.
  3. To restrict content: Select a membership plan from the "Restrict access to members of:" dropdown. Only members of that plan will be able to see the content.
  4. To schedule (drip) content: In the "Release Delay (in days)" field, enter the number of days after a member signs up that they should get access to this content. For example, entering 7 will make the content available one week after their subscription starts.
  5. Save or update the post.

3.3. Managing Secure File Downloads

You can offer protected files that are only accessible to specific members.

  1. Navigate to Secure Files → Add New.
  2. Give your file a title (e.g., "My Awesome E-book").
  3. In the "File Details & Restrictions" box, click Select or Upload File to choose a file from your media library.
  4. In the "Access Restriction" section, check the box next to each membership plan that should have access to this file.
  5. Publish the file.
  6. To display the download link on a post or page, use the [wmp_download] shortcode with the post ID of the secure file. For example: [wmp_download id="123" text="Download Your E-book"].

3.4. Managing One-Time Offers (Upsells/Downsells)

You can create special offers that are presented to users after they purchase a primary plan.

  1. First, create a new membership plan that will serve as your offer. Navigate to Membership Plans → Add New.
  2. In the "One-Time Offer (OTO) Settings" section of the plan editor, check the box for Is this a One-Time Offer?. This will hide it from the main plans page.
  3. To create an upsell: In the "Upsell for Plan" dropdown, select the primary plan that should trigger this offer. After a user buys that primary plan, they will be redirected to this upsell offer.
  4. To create a downsell: In the "Downsell for Plan" dropdown, select the primary plan. This offer will be shown if the user *declines* the main upsell for that primary plan.
  5. Publish the OTO plan.

4. Affiliate Management

4.1. Affiliate Registration & Dashboard

  • How to Apply: Create a page with the [wmp_affiliate_registration] shortcode. Users can visit this page to apply to become an affiliate.
  • How to Track Sales: Create a page with the [wmp_affiliate_dashboard] shortcode. Once an affiliate is approved, they can visit this page to get their unique referral link and see their performance statistics.

4.2. Managing Affiliates (Admin)

Navigate to Subscriptions → Affiliates. Here you can see a list of all affiliates and their status. You can approve or reject pending applications from this screen.

4.3. Paying Affiliates

  1. When an affiliate reaches the minimum payout threshold, they can request a payout from their dashboard.
  2. As an admin, navigate to Affiliates → Payouts to see a list of all pending payout requests.
  3. Process the payment to your affiliate through your chosen method (e.g., PayPal, bank transfer). This part is handled outside the plugin.
  4. Once you have sent the payment, you can mark the payout request as "Completed" in the admin dashboard (future feature). For now, this page serves as a record of all payout requests.

5. Reports & Analytics

The plugin includes a dedicated reporting section to help you track your business's performance.

  • Navigate to Reports from the main admin menu.
  • Here you will find key metrics like Monthly Recurring Revenue (MRR).
  • You can also export your transaction data to a CSV file from this page for further analysis in spreadsheet software like Excel or Google Sheets.

6. Creating Sample Data for Testing

To effectively test your new membership site, you should create some sample data:

  1. Create Plans: Go to Membership Plans → Add New to create at least two different tiers (e.g., "Basic" and "Pro").
  2. Create a Coupon: Go to Subscriptions → Coupons to create a test coupon (e.g., 10% off).
  3. Create a Test User: Go to Users → Add New. Create a new user with the "Subscriber" role.
  4. Simulate a Purchase: Log in as the test user (or use a different browser) and go through the purchase process on your "Plans" page. Use the test coupon and a simulated payment method to create a subscription and transaction record.
  5. Test Affiliate Flow: Have your test user apply to be an affiliate. As an admin, approve their application. Then, use their referral link in an incognito window to simulate another user signing up. Check the affiliate dashboard to see if the conversion was tracked.
